# 異世界ギルドマスターズ + 迷宮大変動 & 迷宮都市の群像劇

## ルールの設定

1. マップの選択
    - 1.基本: 六角形マップ(2-4 人向け)
    - 2.基本: 古墳型マップ(4-5 人向け)
    - 3.基本: チュートリアルマップ(2-5 人向け)
    - 4.拡張: 迷宮大変動 表面
    - 5.拡張: 迷宮大変動 裏面
2. 変動迷宮使用の有無
    - 1.使用する(迷宮大変動マップの場合強制)
    - 2.使用しない
3. 追加ルール「移り変わる迷宮」の使用有無（妨害カードの導入）
    - 1.使用する
    - 2.使用しない
4. 追加ルール「ギルドの掲示板」の使用有無
    - 1.使用する
    - 2.使用しない
5. 長時間ゲームルール「汚染領域」の使用有無
    - 1.使用する
    - 2.使用しない
6. 「豊かな季節」ターントラックの使用有無
    - 1.使用する
    - 2.使用しない
